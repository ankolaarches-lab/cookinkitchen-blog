# Content Templates for Proven Pantry

## Review Template (Minimum 800-1000 words)

**Structure:**
```
## [Product Name] Review
### Quick Verdict: ★ X.X [one sentence summary]

### Introduction (50-75 words)
[Hook + what this product is + who it's for]

### First Impressions (100-150 words)
[Unboxing, build quality, initial thoughts]

### Performance / Testing Notes (200-300 words)
[Real-world use cases, how it performed]

### Specs Table
| Feature | Value |
|---------|-------|
| [Spec 1] | [Value] |
| [Spec 2] | [Value] |

### Pros
- [Pro 1]
- [Pro 2]
- [Pro 3]

### Cons
- [Con 1]
- [Con 2]

### Alternatives to Consider
- [Alternative 1]: [brief why]
- [Alternative 2]: [brief why]

### Final Verdict (75-100 words)
[Summary recommendation, best for whom, value rating]

### Where to Buy
[Amazon link with affiliate tag]
```

```

### TL;DR: [One sentence summary]

### Introduction (100-150 words)
[Hook + why this matters + what reader will learn]

## Section 1: [Topic]
[Detailed content with specific advice, examples]

## Section 2: [Topic]
[Detailed content...]

## Section 3: [Topic]
[Detailed content...]

## Common Mistakes to Avoid (if applicable)
- [Mistake 1]: [How to fix]
- [Mistake 2]: [How to fix]

## Pro Tips
- [Tip 1]
- [Tip 2]

## Recommended Products (if applicable)
- [Product 1]: [Why]
- [Product 2]: [Why]

## Conclusion
[Summary + call to action]
```
